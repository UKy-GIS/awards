RULES

Four games with one practice game

Each game has three rounds. Place a marker on the map where you think the location is. The closest to the actual location wins.

Three teams

Each team can have as many players as desired. The average of all players will be the average of the team's score.

Smacktalk is allowed

But using any other resource to discover the location is not.
